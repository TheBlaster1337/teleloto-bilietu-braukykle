# Teleloto bilietų braukyklė
Norintiems braukyti teleloto bilietus kompiuteryje.

## Funkcijos
- Įkėlimas naudojant JSON failą.
- Braukymas spaudžiant ant skaičių bilietuose
- Įvesto skaičiaus išbraukimas visuose bilietuose.

## Instaliavimas
```
npm install
```
```
npm run dev
```
